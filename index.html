iOS: главный экран и виджет

Что реализовано
WebApp можно открыть как PWA с главного экрана iPhone. Настоящий iOS-виджет работает через отдельный WidgetKit-код из архива и читает тот же Sync API, поэтому данные Telegram и iOS совпадают.
1. В Telegram откройте настройки синхронизации, укажите Sync API и нажмите «Связать Telegram ID».
2. Скопируйте ссылку для iPhone, откройте её в Safari: Поделиться → На экран «Домой».
3. Для настоящего виджета соберите папку ios/TaskPlannerIOS в Xcode, установите приложение и вставьте тот же токен/ссылку.
